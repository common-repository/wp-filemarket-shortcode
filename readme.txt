The FileMarket Shortcode for WordPress lets you easily include a "Buy it now" button you previously configured in the http://filemarket.online digital goods marketplace

== Description ==

FileMarket online lets you sell file downloads with zero configuration on your side. You just specify your PayPal email, your price and your file and we will take care of all the technical details for you. Visit https://filemarket.online and start selling! 


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/filemarketshortcode` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the shortcode as follows:
[filemarket id="yV67MdRoP21hKCCdNoIs" return="http://filemarket.online/"]
Where:
The `id` attribute you get when you add your file to the filemarket
And the `return` is the address you want to redirect the users after successful payment
